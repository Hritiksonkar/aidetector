# Frontend (Vite + React + Tailwind)

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Backend base URL default: `http://localhost:5000` (override via `.env` using `VITE_API_BASE_URL`).

To configure:
- Copy `frontend/.env.example` to `frontend/.env`
- Set `VITE_API_BASE_URL` if frontend and backend are on different origins
